# Kevin Wangombe - Personal Portfolio

Welcome to the personal portfolio website of Kevin Wangombe, a Fullstack Web Developer. This website showcases my skills, experience, and featured projects. It is designed to be fully responsive, user-friendly, and modern.

**[🌐 View the Live Website Here](https://kevinwangombe.github.io/website-portfolio/)**

## Features

-   **Hero Section:** A brief introduction with a quick overview of what I do.
-   **About Me:** Information about my background, passion for web development, and professional experience.
-   **Skills:** A visual breakdown of my technical skills categorized into Frontend and Backend development.
-   **Featured Projects:** A showcase of some of my highlighted projects, including an E-Commerce Platform, a Social Media Dashboard, and a Task Management App.
-   **Contact:** A dedicated section to get in touch with me via email or connect on social platforms like GitHub and LinkedIn.
-   **Responsive Design:** The layout adjusts seamlessly across mobile, tablet, and desktop devices.
-   **Smooth Scrolling:** Enables fluent navigation between sections on the single-page layout.
-   **Scroll Animations:** Interactive element reveals as you scroll down the page.

## Technologies Used

This project is built using core web technologies, ensuring high performance, accessibility, and clean architecture without relying on heavy frameworks for the frontend interface:

-   **HTML5:** For the structural foundation of the site.
-   **CSS3:** For styling, including responsive layouts with Flexbox/Grid, custom variables, and modern visual effects (like glassmorphism and gradient text).
-   **JavaScript (ES6):** To handle mobile navigation, smooth scrolling, scroll reveal animations, and dynamic footer year updates.
-   **Lucide Icons:** A beautiful and consistent icon set used throughout the UI.
-   **Google Fonts:** Utilizing 'Inter' and 'Poppins' for clean and modern typography.

## How to Run Locally

Since this is a static website, no build processes or complex servers are required to run it locally.

1.  Clone this repository or download the source code.
2.  Navigate to the project directory.
3.  Open the `index.html` file in any modern web browser.

Alternatively, you can run a local server (e.g., using Python, Node.js, or VS Code Live Server extension) to serve the files:

```bash
# Using Python 3
python3 -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.
